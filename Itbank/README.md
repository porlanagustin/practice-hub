# Bank Project

## Overview
This project is a banking application developed using Django, Python, JavaScript, HTML, and CSS. It aims to provide users with a seamless experience for managing their banking needs, including account management, transfers, and deposits.

## Technologies Used
- **Backend**: Django, Python
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQLite

## Features
- User authentication and authorization
- Account creation and management
- Transfer funds between accounts
- Deposit funds
- Transaction history
